# ESTE É A PRIMEIRA AULA DE GIT E GITHUB
## ABAIXO LISTO OS PRINCIPAIS COMANDOS

* git init
* git status
* git add arquivo.txt
* git commit -m "Meu primeiro commit"
* git log
* git log -p
* git log --pretty=format "%h - %ar : %s"
* git log --since=2.days
* git add .gitignore
* touch arquivo.txt
* git reset HEAD arquivo.txt
* git checkout  heach
* git checkout 'master'
* git reset HEAD~1 --soft
* git reset HEAD~2 --hard
